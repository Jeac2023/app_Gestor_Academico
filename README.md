
# Dashboard de Ventas Académicas 📚

Este proyecto es un panel de control interactivo para visualizar y gestionar los trabajos académicos registrados en un archivo Excel.

## 🚀 ¿Qué puedes hacer con esta app?
- Subir tu archivo Excel con múltiples hojas de diferentes grupos.
- Ver resumen de ingresos, adelantos y cantidad de trabajos.
- Ver distribución de estados (pendiente, entregado, etc.).
- Ver rankings de desarrolladores.
- Ver tareas pendientes ordenadas por fecha.
- Revisar observaciones registradas.

## 🛠️ Requisitos
El archivo `requirements.txt` incluye:
```
streamlit
pandas
plotly
openpyxl
```

## ▶️ Cómo usar en Streamlit Cloud
1. Crea un repositorio privado en GitHub.
2. Sube los archivos `app.py`, `requirements.txt` y `README.md`.
3. Ve a [Streamlit Cloud](https://streamlit.io/cloud), inicia sesión con tu cuenta de GitHub.
4. Crea una nueva app seleccionando el repositorio y el archivo `app.py`.
5. Configura permisos si deseas mantener la app privada.

¡Listo! Tu dashboard estará disponible online.
